# DateLog

DateLog is a dating memory app for couples. Couples can record a date route on a map, attach places, photos, and notes, and revisit previous date records later.

## Current Phase

Frontend MVP scaffolding.

This workspace is focused on frontend development. Backend implementation is handled separately by another teammate, and this repo keeps API expectations as a contract draft.

## MVP Goal

Build an app where a couple can draw one day's date route on a map, save places, photos, and notes on that route, and view the record later.
